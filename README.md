# Amazon Clone

> My groundbreaking Nuxt.js project

## Build Setup

```bash
# install dependencies for all folders
$ npm run install

# serve with hot reload at localhost:3000
$ npm run serve

# client at localhost:3333
$ npm run dev

# admin at localhost:8000
$ npm run dev

```
